# todo-list 

> Todo List project created using React and Material UI Libary. 

## Prerequisites

- [nodejs](https://nodejs.org/en/) 
- [yarn](https://www.boost.org/)

## Install

```sh
npm install
```
or 
```sh
yarn install
```

## Run

In the directory of the repo:
```sh
npm start
```
or
```sh
yarn start
```
