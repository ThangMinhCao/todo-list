# todo-list 

> Todo List project created using React and Material UI Libary. 

### :link: [Try it out](https://todo-list-4bdd6.web.app/)

## Install

```sh
npm install
```
or 
```sh
yarn install
```

## Run

In the directory of the repo:
```sh
npm start
```
or
```sh
yarn start
```
